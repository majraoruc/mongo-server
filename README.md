# mongo server
Simple Mongo Server

Here's a simple mongo server implementation

# How to set it up

1. Clone the repository https://github.com/majraoruc/mongo-server.git
2. Open in VS Code or any other editor of your choice (I prefer VSCode)
3. ```  npm install  ```
4. ```  npm start  ```
